# Zealot 服务架构

运行 Zealot 依赖如下服务才能保证正常运行

- 支持 `amd86`、`arm64` 架构的主流 Linux 系统
- Zealot 提供 Web 服务、API 服务和后台任务服务
- 本地/云端存储用于持久化存储上传应用和图标
- PostgreSQL 数据库和缓存服务
