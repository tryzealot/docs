# 用户权限

Zealot 提供四种用户权限：

- 游客
- 普通用户
- 开发者
- 管理员

## 权限明细

`App`:<br />浏览应用内的网络钩子 | ✓/✕ | ✓ | ✓ | ✓
`App`:<br />上传/安装/下载应用 | ✓ | ✓ | ✓ | ✓
`App`:<br />创建/编辑/删除应用 |  |  | ✓ | ✓
`App`:<br />创建/编辑/删除应用的类型或渠道 |  |  | ✓ | ✓
`App`:<br />删除指定的应用版本和上传的应用文件 |  |  | ✓ | ✓
`App`:<br />测试应用的网络钩子 |  |  | ✓ | ✓
`App`:<br />启用或禁用网络钩子 |  |  | ✓ | ✓
